# LeetCode Solutions

Daily LeetCode solutions in JavaScript with Big O time and space complexity analysis. One problem per day.

## Structure

Each file is named: `[problem-number]_[problem-name].js`

## Format

Each solution includes:

- My original solution
- Optimal solution
- Time and space complexity analysis

## Progress

- Day 1: Merge Strings Alternately - O(n) time O(n) space

- Day 2: Greatest Common Divisor of Strings - Approach 1: O(min(m,n)×(m+n)) time O(min(m,n)) space | Approach 2: O(m+n) time and space
